# Show Context: The Debug Log

## Identity
- **Name / Tagline:** The Debug Log — Making sense of software, one bug at a time
- **Format:** Two hosts, deep-dive, 30-45 min, weekly
- **Target audience:** Working software engineers, mid-level to senior
