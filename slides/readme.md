# AI Podcast Workshop — Slides

Slidev deck for the **AI Podcast Studio** workshop, built on the
`WorkshopForAgentic` module from `edgeai-for-beginners`.

## Run locally

```bash
npm install
npm run dev
```

Opens at <http://localhost:3030>.

## Export

```bash
npm run export      # PDF
npm run build       # static SPA -> dist/
```

## Editing

All content lives in [slides.md](./slides.md). Slides are separated by `---`.
